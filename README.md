# Credit_Risk

## Overview

The purpose of this analysis was to analyze how effective different machine learning models are at predicting credit risk.

## Results

* Naive Random Oversampling: Accuracy (0.632), Precision (0.99), Recall (0.55)
* SMOTE Oversampling: Accuracy (0.624), Precision (0.99), Recall (0.55)
* Undersampling Cluster Centroids: Accuracy (0.624), Precision (0.99), Recall (0.67)
* Combination (Over and Under Sampling) SMOTEENN: Accuracy (0.608), Precision (0.99), Recall (0.55)
* Balanced Random Forest Classifier: Accuracy (0.918), Precision (0.99), Recall (0.92)
* Easy Ensemble AdaBoost Classifier: Accuracy (0.92), Precision (0.99), Recall (0.92)


## Summary

Based on the accuracy the different models, the best model to proceed with would be Easy Ensemble AdaBoost Classifier.
